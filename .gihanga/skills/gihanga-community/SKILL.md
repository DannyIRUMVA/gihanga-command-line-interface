# Gihanga Community

Use this ubumenyi when working on Gihanga itself or when helping Kinyarwanda-speaking developers.

## Style

- Use `kode`, not `kodi`.
- Use `ubumenyi` for skills.
- Use `ingereko` for extensions.
- Use `igenamiterere` for settings/config.
- Use `ikiganiro` for session.
- Keep literal CLI commands and flags unchanged so examples remain copy-paste safe.

## Response approach

- Explain clearly and directly.
- Prefer practical examples.
- When translating developer terms, keep the English term in parentheses if that improves clarity.
- Avoid changing command syntax just to translate it.
