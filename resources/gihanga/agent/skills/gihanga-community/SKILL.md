# Gihanga Community

Use this ubumenyi when working on Gihanga itself or when helping Kinyarwanda-speaking developers.

Ubu bumenyi bushyirwa muri `~/.gihanga/agent/skills/gihanga-community/` na installer ya Gihanga. Bujyana na data file:

`~/.gihanga/agent/data/kinyarwanda-keywords.json`

## Sources used for the initial keyword pack

- Rwanda Cultural Heritage Academy ICT terminology examples reported by Technext: https://technext24.com/news/rwanda-launch-local-tech-dictionary/
- LearnEntry common Kinyarwanda vocabulary examples: https://www.learnentry.com/english-kinyarwanda/1000-most-common-kinyarwanda-words
- 1000 Most Common Kinyarwanda Words examples: https://1000mostcommonwords.com/1000-most-common-kinyarwanda-words

Do not claim the local data file is a complete official dictionary. Treat it as a practical starter glossary for CLI copy, prompts, and beginner explanations.

## Gihanga style rules

- Use `kode`, not `kodi`.
- Use `ubumenyi` for skills.
- Use `ingereko` for extensions.
- Use `igenamiterere` for settings/config.
- Use `ikiganiro` for session.
- Use `/kwinjira` for login.
- Use `/sohoka` for quit.
- Use `/continue` with the Kinyarwanda meaning `komeza`.
- Keep literal CLI flags and command syntax unchanged when copy-paste compatibility matters.
- When translating developer terms, keep the English term in parentheses if that improves clarity.

## Response approach

- Explain clearly and directly.
- Prefer practical examples.
- For beginners, define technical terms before using them heavily.
- If a Kinyarwanda term may confuse developers, write: `Kinyarwanda term (English term)`.
- Avoid inventing official terminology. If unsure, say it is a practical Gihanga wording, not an official standard.
